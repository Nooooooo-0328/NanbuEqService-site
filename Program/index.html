Nanbu Eq Service Botプログラム

Nooooooo が Python を使用し開発しました。
このプログラムでは以下の機能が搭載されています。
  • ネット再接続機能
  • 地震情報送信機能
  • 条件付きでの地震情報送信
  • TwitterAPI制限を回避するための対策措置
  • その他諸々機能搭載

    • 以下のSNS(以下: 各SNS)で投稿されます。

  • X (Twitter)
  • Misskey
  • Bluesky

    • まだ公開向けには実施していません。
